# EaseFlows are built for outcomes

For a household, a capability only matters if it can complete a class of work reliably.

## Beyond single-step skills

An EaseFlow includes:

- Goal
- Trigger
- Roles and permissions
- Confirmation rules
- Recovery paths

## The advantage

This makes household execution teachable. Instead of repeating the same coordination work manually, families can refine the way work gets done and reuse it over time.
